# BAMOE Developer Tools Extension - Basic Forms

This directory contains basic forms as would be generated by the BAMOE Developer Tools VS Code extension, following IBM documentation.

## Extension Workflow (IBM Documentation)

1. **Install Extension**: BAMOE Developer Tools in VS Code
2. **Open BPMN**: Right-click on user task in BPMN diagram
3. **Design Form**: Select "Design Form" > "Basic Form"
4. **Configure Fields**: Add input fields with basic validation
5. **Generate**: Extension creates HTML forms automatically
6. **Deploy**: Forms are served from META-INF/resources/forms/

## Generated Forms

- `dcss-children-form.html` - Basic children count input
- `dcss-dependent-form.html` - Basic dependent information form

## Key Features (Extension Generated)

- ✅ Basic HTML/CSS/JavaScript forms
- ✅ Simple validation and error handling
- ✅ Automatic BPMN process integration
- ✅ REST API calls to process endpoints
- ✅ Basic responsive design

## Limitations vs Custom Forms

- No advanced UI components (Patternfly)
- Basic styling and UX
- Limited validation rules
- No advanced business logic integration

## Access

- Dev UI: http://localhost:8080/q/dev-ui/extensions
- Direct: http://localhost:8080/forms/[form-name].html
